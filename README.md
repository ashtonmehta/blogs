# Blogs

[VIEW PROJECT HERE!](https://ashtonmehta.github.io/blogs/)

## Description
A simple blog website where users can create, edit, and delete posts. Users can also sign in with their Google account.

## Technologies
- Vite
- React
- TypeScript
- Tailwind CSS
- Firebase Authentication
- Firebase Firestore
